# Product Vocabulary

**DNA-VOCAB-001** · Volume 24 Ch. 26

Use these terms consistently in UI, docs, and support.

| Term | Use | Avoid |
|------|-----|-------|
| Trip | A business journey from start to end | Journey, drive, travel event |
| Receipt | Proof of purchase | Ticket (unless parking/transit type) |
| Expense | A cost record | Charge (unless line item) |
| Report | Generated mileage/expense document | Export (as noun) |
| Business | User's work entity | Company (in UI) |
| Vehicle | Car/truck used for trips | Auto, car (prefer Vehicle in settings) |
| Mileage | Distance driven | Miles traveled (labels: **Miles**) |
| Dashboard | Home summary screen | Home, overview |
| Attach | Link receipt to trip | Associate, connect |

Add terms here before introducing new UI labels.
